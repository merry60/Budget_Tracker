A personal budget tracker app built with Flutter for Pakistani users.

Most budget apps use dollars and don't have categories that make sense
for Pakistan. I wanted something simple that works in PKR with local
categories like Rashan, Bijli, and Safar. Also wanted to practice
Flutter so this was a good project to build.

## Features
- Add and track daily expenses in PKR
- Pakistani categories (Rashan, Bijli, Safar, Gas Bill, Mobile Balance etc.)
- Set a monthly budget — get a warning when you are close to the limit
- View expense history and filter by month
- Pie chart and bar chart to see where money is going
- Dark mode
- Works fully offline — no internet needed, no account required

## Tech Used
- Flutter & Dart
- Hive — for saving data locally on the device
- Provider — for state management
- fl_chart — for the charts
- SharedPreferences — for storing settings




## What I Learned
- How to use Hive for local storage in Flutter
- How Provider and ChangeNotifier work
- How to implement charts using fl_chart
- Basic app architecture with screens, models and providers
